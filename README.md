
# Go-rengan
Go codebase with oil

## Stack
- Chi (net/http)
- MongoDB
- Opentelemetry
- Uptrace

## Run

Run Uptrace with docker compose

```bash
  docker compose up -d
```

Start the server

```bash
  make run
```

## Unit Test
Run Unit testing
```bash
  make test
```

Run Coverage
```bash
  make test/cover
```
